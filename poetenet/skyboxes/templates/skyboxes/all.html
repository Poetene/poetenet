Skybox

A skybox is a set of images (6 in total). Each image is the texture for one of the surfaces of the skybox cube.

Example URL: {{ STATIC_URL }}res/skyboxes/SunSet_left.jpg
In this example, the prefix is "SunSet" and the surface is "left". So the format of the URL is as follows:
{{ STATIC_URL }}res/skyboxes/prefix_surface.jpg
surface
The surfaces are "left", "right", "up", "down", "front", "back"

prefix
Here's a set of prefixes for skyboxes that you can use:

    {% for skybox in skyboxes %}
  • {{ skybox.prefix }}
    • {% endfor %}
